# Data-mining-prediction
生活方式时间分配对寿命影响的数据挖掘与预测

本研究以{生活方式与寿命}为核心议题，通过全流程数据挖掘方法，系统探究了个体日常时间分配模式与寿命之间的关联规律。研究采用包含1万名个体的生活方式与寿命信息数据集，首先通过探索性数据分析（EDA）可视化呈现了各生活方式特征（工作、睡眠、锻炼时长等）的分布、相关性及异常模式。

在无监督学习阶段，运用 K-Means 聚类算法，基于个体的工作、睡眠和锻炼时长划分出4个具有显著差异的生活方式群体。通过 PCA 降维技术将高维特征映射至二维空间，直观验证了聚类结果的有效性与群体间的区分度。分析表明，不同生活方式群体的寿命水平存在统计学显著差异，其中{健康均衡型}群体展现出最长的预期寿命，而{高风险生活方式型}群体的寿命显著偏低。

在监督学习阶段，本研究构建并对比了两种主流预测模型：随机森林（一种基于决策树集成的稳健模型）与神经网络（一种多层感知机，MLP，用于捕捉复杂非线性关系）。通过将数据集划分为训练集与测试集（8:2比例），对模型进行训练与评估。结果显示，神经网络模型在测试集上取得了(R^2=0.558) 的预测精度，略优于随机森林模型（R^2=0.532），表明其在拟合生活方式与寿命间复杂的、可能存在收益递减效应的非线性关系方面表现更佳。

最后，通过模型持久化技术将训练好的预测模型保存，实现了对新个体寿命的快速预测。本研究不仅为个体提供了量化的生活方式优化指南，也为公共卫生政策制定和健康管理干预策略提供了坚实的数据支撑。
