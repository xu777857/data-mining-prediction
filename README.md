# 生活方式时间分配对寿命影响的数据挖掘与预测
# 简介
在公共卫生与健康管理的研究体系中，个体生活方式作为影响健康结局与寿命的核心可控因素，其与健康指标的关联解析始终是领域内的研究热点之一。随着现代社会快节奏发展与职场竞争加剧，“超时工作、睡眠不足、缺乏锻炼” 等失衡生活方式逐渐成为职场人群的常态 ——《中国职场人群健康报告》数据显示，超 60\% 的职场个体每日工作时长超过 8 小时，35\% 人群日均睡眠不足 6 小时，这类行为模式的异化已成为慢性疾病发病及寿命缩短的潜在诱因，也使得 “生活方式与寿命的量化关联” 成为兼具现实紧迫性与应用价值的研究命题。

然而，当前针对生活方式与寿命的研究仍存在显著局限：其一，现有研究多聚焦单一因素的独立影响（如睡眠时长或锻炼频率对健康的作用），尚未充分揭示工作、睡眠、锻炼等日常时间分配维度的综合交互效应，难以匹配 “时间分配总和为 24 小时” 的真实生活模式；其二，研究多以整体样本为分析对象，缺乏对生活方式群体差异的系统性划分，无法针对 “过劳型”“均衡型” 等不同行为模式的群体提供精准化健康建议；其三，现有成果多停留在关联描述层面，缺少可落地的量化预测模型，难以实现 “输入个体时间分配数据、输出寿命预测值” 的个性化分析。

基于此，本研究以包含 1 万名个体日常时间分配（工作、休息、睡眠、锻炼）与寿命信息的数据集为研究载体，借助完整数据挖掘流程（探索性数据分析、聚类分群、多模型预测等）展开实证分析。

# 数据来源
本文从kaggle平台获取工作、生活平衡与寿命数据集。该综合数据集包含1万名个体信息，探讨年龄、职业、每日时间分配（工作、休息、睡眠、锻炼）与寿命之间的关系。这些数据旨在支持回归建模和分析生活方式选择如何影响寿命。同时该数据集捕捉了日常活动总和约为24小时的真实模式，并包含代表极端工作或休息模式的异常值。这使其成为探索健康结果中非线性关系和收益递减的理想选择。


# 如何运行代码

1. 安装Python库/直接运行pip install -r requirements.txt

你的电脑需要先安装numpy、pandas、matplotlib、seaborn、scikit-learn、joblib这几个库。

可以在电脑的终端或命令行输入：

pip install pandas scikit-learn matplotlib seaborn scikit-learn joblib

3. 下载代码和数据
点击本页面绿色的Code按钮，选择Download ZIP，解压到你的电脑。

4. 运行预测脚本
在解压后的文件夹里，打开终端或命令行，输入：
python predict_lifespan_rf.py
等待片刻，就会输出模型的预测准确率结果。


# 评价
优点：整合无监督聚类与监督预测的全流程数据挖掘方法，既实现了生活方式群体的分类，也完成了寿命的精准预测；通过多模型对比提升了预测结果的准确性，同时结合特征重要性分析增强了结果的可解释性；研究成果既可为个体健康管理提供量化指南，也能为公共卫生政策制定提供数据支撑，适用场景广泛。

缺点：数据集仅覆盖生活方式类特征，未纳入遗传、医疗史等关键健康因素，可能限制模型的泛化性；神经网络模型存在一定过拟合风险，需进一步通过交叉验证优化；聚类数的选择依赖肘部法则，存在一定主观性，后续可结合轮廓系数等指标进一步验证。
